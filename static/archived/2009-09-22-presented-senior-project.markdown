วันนี้สอบโปรเจ็คเรียบร้อยแล้วครับ ทุกอย่างผ่านไปได้ด้วยดี ทำได้ตามที่คิดไว้ .. อาจารย์คณะกรรมการก็มีซักถามเล็กน้อยครับ กลุ่มอื่นมักจะได้รับ scope งานเพิ่มจากอาจารย์ แต่กลุ่มของผมกลับไม่เป็นแบบนั้นเลย
